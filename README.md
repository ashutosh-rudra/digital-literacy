About This Project

The main idea of this project was to understand how to use digital tools properly and responsibly as a student. It’s not just about technical skills, but also about things like communication, online safety, and building a professional presence.

All the work is divided into 5 tasks, and each task focuses on a different aspect of digital literacy.

Project Structure

Here’s how I’ve organized everything in this repository:

task-1-presentation/ → Infographic on digital literacy
task-2-portfolio/ → Screenshots of my profiles (GitHub, LinkedIn, etc.)
task-3-platforms/ → Coding platform activity + Google Form
task-4-email-etiquette/ → Email drafts and social media checklist
task-5-cybercrime/ → Case study and safety tips
report/ → Final project report
Task Summary
Task 1 – Digital Literacy Infographic

For this task, I created an infographic to explain what digital literacy is and why it matters. I used an online design tool to make it simple and easy to understand. It covers topics like safe internet usage, useful tools for students, and maintaining a professional online presence.

Task 2 – Student Digital Portfolio

In this task, I set up and improved my profiles on different platforms like GitHub and LinkedIn. The goal was to start building a professional identity online. I also learned how these platforms can help in internships and future opportunities.

Task 3 – Platforms Exploration

This task had two parts:

I solved a beginner problem on a coding platform to get familiar with how it works.
I created a Google Form as a digital literacy quiz.

This helped me understand both coding practice and collaboration tools.

Task 4 – Email Etiquette

Here, I wrote two professional emails:

Requesting an assignment extension
Applying for an internship

I also made a checklist of do’s and don’ts for social media use. This task made me realize how important tone and clarity are in communication.

Task 5 – Cybercrime Awareness

For the last task, I worked on a cybercrime case study and created a prevention checklist. It mainly focuses on how such incidents happen and how we can avoid them, especially as students.

What I Learned

Overall, this project was quite useful. I learned:

How to present myself professionally online
How to communicate properly through emails
Basic awareness about cyber safety
How different platforms can be used for learning and growth

It also made me more confident in using digital tools in a practical way.
